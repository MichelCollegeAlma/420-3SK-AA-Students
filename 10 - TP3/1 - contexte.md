# TP3 - Serveur Mail `formation.lan` [10 %]

## 1/6 - Contexte

Votre département informatique souhaite mettre en place un serveur mail interne pour faciliter la communication entre les différents membres de l'équipe.

Actuellement, tout le monde utilise des services externes (Gmail, Outlook) pour communiquer, mais le département souhaite :
- Avoir un contrôle complet sur les données
- Former les étudiants sur l'administration d'un serveur mail
- Créer un environnement de test sécurisé
- Apprendre les protocoles SMTP, IMAP et la gestion des utilisateurs

Le serveur mail iRedMail est maintenant la solution qui s'impose.
