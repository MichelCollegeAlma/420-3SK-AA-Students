# 2.2 - Examen1 - ServeurWeb_LAMP

Vous devez configurer un serveur LAMP tel que vue en cours.

Lorsque le serveur sera fonctionnel, installez un site Wordpress

## Configuration spécifique du serveur

Voir [1.1 - Contexte - Directives générales.md](1.1%20-%20Contexte%20-%20Directives%20g%C3%A9n%C3%A9rales.md)

## Service à installer
- Apache
- MySQL
- Php
- PhpMyAdmin

*La liste de ces services n'est pas exhaustive, il peut y avoir des dépendances.*

## Tâches après installation
- Installation et configuration du site Wordpress `hohoho.votreDA.lan`.
- Le site est basé sur la thématique de Noël et un thème de Noël gratuit.
  - Exemples de thèmes gratuit: 
    - https://wordpress.org/themes/christmas-holiday/
    - https://wordpress.org/themes/christmas-bell/
    - https://wordpress.org/themes/christmaspress/
 
## Validations et tests
- Accès à PhpMyAdmin
- Accès au site Wordpress via l'adresse `hohoho.votreDA.lan`
- Fonctionnalité du site
  - Thématique de Noël 
  - Téléversement d'une image
  - Ajout d'informations prouvant son fonctionnement

*Liste non exhaustive, d'autres tests peuvent être nécessaire*
