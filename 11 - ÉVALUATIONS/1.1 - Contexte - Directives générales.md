# 1.1 - Contexte de l'examen et de l'ETC
# Directives générales

Avec l'objectif de simplifier votre travail, l'examen et l'ETC se réaliseront simultanémement.

Mise-à-part l'implantation du serveur DNS, il n'y a pas d'ordre pour l'implantation et la configuration des serveurs et la réalisation de l'examen n'est pas obligatoire à la réussite de l'ETC.

**En clair, commencez par le serveur DNS et ensuite c'est comme vous voulez.**

Lisez tout les documents avant de commencer pour bien comprendre ce qui est demandé.

L'enseignant peut répondre aux questions de compréhension des énoncés, mais aucune sur la matière.

Vous avez droit à tout, mais l'évaluation doit être faite par vous seul.

Les critères d'évaluation et les modalités de remise sont dans les documents:
- [2.3 - Examen1 - evaluation_remise.md](2.3%20-%20Examen1%20-%20evaluation_remise.md)
- [3.4 - ETC -  evaluation_remise.md](3.4%20-%20ETC%20-%20%20evaluation_remise.md)

## Machines virtuelles et configuration des serveurs
Tout les serveurs sans exception on les configurations suivantes:
- Proxmox
  - Resource Pool: Utilisez le votre
  - IOS image: ubuntu-24.04.3-live-server-amd64.iso
  - CPU Cores: >= 2
  - Memory: >= 4000
  - Network model: VirtIO (paravirtualized)
- Linux
  - OS: Ubuntu server 24 LTS 
  - Login: test
  - Mot de passe: test (lorsque possible)
  
Les configurations suivantes sont différentes sur chacun des serveurs:
- Proxmox
  - VM ID (Utilisez la forme Plage**Y** où vous choisissez **Y**)
  - VM name (Utilisez **Nom de la VM** dans le **Schéma d'adressage**)
- Linux
  - Nom d'hôte (Utilisez **Nom d'hôte** dans le **Schéma d'adressage**)
  - IP fixe (Utilisez **IP fixe** dans le **Schéma d'adressage**)
